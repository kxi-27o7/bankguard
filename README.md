# BankGuard

Machine learning-based mobile banking fraud detection system using Random Forest.

## Installation

Clone repository:

```bash
git clone https://github.com/kxi-27o7/bankguard.git
cd bankguard
```

## Download Model

The trained model is not included in this repository because it exceeds GitHub's file size limit.

Download the model here:

[(https://drive.google.com/drive/folders/1kgHz4LNbMl-mfm1qY7pa1EonwPWVvraR?usp=sharing)]

After downloading, place the model file inside:

```plaintext
ml/model/
```

## Run Project

```bash
python backend/app.py
```
